Backend for the free food service.

## Quick Start

```bash
# Install dependencies
npm install

# Run mocha (nodejs) and jest (reactjs) tests
npm run test
# Run tests with coverage
npm run test-cov

# Run server from root
npm run start

# Run server from src/client
npm run start
```
